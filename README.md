Un juego arcade tipo Space Shooter con tem√°tica Neon Cyberpunk, hecho con Expo.

## Como correr

1) Instala dependencias
   - `npm install`
2) Alinea versiones con SDK 54 si ves errores
   - `npx expo install --fix`
3) Levanta el servidor para iPhone con Expo Go
   - `npm run start`

## Controles

- Arrastra el dedo en el area de juego para mover la nave.
- La nave dispara automaticamente.
- Esquiva los enemigos rojos y destruyelos con tus laseres para sumar puntos.
